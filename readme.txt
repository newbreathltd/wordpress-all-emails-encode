A simple plugin encrypt e-mail addresses on that were not available by robots collecting data.

== Description ==

Simple plugin encodes e-mail that they can not be retrieved by robots. Robots do not see addresses e-mail on the website.

E-mail encoded is available on the website, then ask Ajax are all decoded. For a user with JavaScript enabled address is clickable normally.

Noteworthy variable encryption key, so that each page refresh will generate a new sequence.

Plugin encrypt e-mail addresess both in page content and also in a widget.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/NB-Email-Anti-Scraper` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= What email addresses are encoded? =

Encrypted all e-mail addresses. Both entered the body of the post / page and typed in widgets.

= E-mail addresses stored in files in the template will also be encrypted? =

No, only the elements in the content and widgets are encrypted.
